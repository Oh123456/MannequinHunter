# 본 프로젝트 코딩 표준

- 클래스 함수,변수 위치

```
class
{
   GENERATED_BODY

   static, const

   class struct enum

   member function

   UFUNCTION

   virtual, override

   member variable

   UPROPERTY
}
```
- 접근 지정자
  - public, protected, private 순서로 작성한다.
- 변수명은 소문자로 시작한다.
- 함수명은 대문자로 시작한다.
- 클래스, 구조체, enum, 은 언리얼 엔진 코딩 표준을 따른다.
